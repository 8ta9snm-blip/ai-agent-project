# AI Agent Logic - منطق وكيل الذكاء الاصطناعي

## الوصف

هذا المشروع يحتوي على منطق وكيل ذكاء اصطناعي (AI Agent) مصمم لإدارة عملية التوظيف والتسجيل والتدريب للمضيفات الجدد بشكل آلي.

## المحتويات

- `ai_agent_logic.py`: الملف الرئيسي الذي يحتوي على منطق الوكيل والنصوص والمراحل

## المتطلبات

- Python 3.11 أو أحدث
- لا توجد مكتبات خارجية مطلوبة (يستخدم المكتبات القياسية فقط)

## التشغيل

```bash
python3 ai_agent_logic.py
```

## الميزات

- إدارة 6 مراحل للتوظيف والتدريب
- نصوص محادثة جاهزة باللغة العربية
- بنك أسئلة وأجوبة شامل
- محاكاة لمنطق الوكيل
